# Ekonomii Quiz

A web-based quiz application for testing knowledge in Economics (Ekonomie). This interactive quiz features 425 questions divided into 8 groups, allowing users to practice and assess their understanding of economic concepts.

## Features

- **Multiple Quiz Modes**:
  - **Random**: Answer a selected number of random questions (up to 400).
  - **All Questions**: Go through all 425 questions sequentially.
  - **Groups**: Focus on specific groups (1-8) for targeted learning.

- **User-Friendly Interface**: Clean, responsive design with progress tracking, score display, and navigation controls.
- **Czech Language**: All text and questions are in Czech, making it suitable for Czech-speaking students or learners.

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla)

## How to Use

1. Clone the repository.
2. Open index.html in your web browser.
3. Select your preferred quiz mode and settings.
4. Start the quiz and answer the questions.
5. Use navigation buttons to move between questions or reset the quiz.
